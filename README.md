# Mohamad Nazrin Napiah - System Integrator Portfolio

This is the professional portfolio of Mohamad Nazrin Napiah, pivoting towards **System Integration**, **IoT Infrastructure**, and **Edge AI**. It is built using **React + Vite** and uses a bespoke Vanilla CSS design system.

## How to Run the Website Locally

To run the web application on your local machine, follow these simple steps:

1. **Open your terminal** in the root of the project directory (`d:\Coding\portfolio`).
2. **Start the development server** by running the following command:
   ```bash
   npm run dev
   ```
3. **View in Browser**: Once the server starts, it will give you a local URL (usually `http://localhost:5173/`). Open that URL in your web browser to see your new website live!

## Making Edits
- The main components (like the Hero section, Services, and Interactive Demo) are located in the `src/components/` folder.
- If you make any changes to the code while `npm run dev` is running, the browser will automatically reload to show your updates instantly!

## Building for Production
When you are ready to host the website online (e.g., on GitHub Pages, Vercel, or Netlify), you can create an optimized production build by running:
```bash
npm run build
```
This will compile everything into a `dist/` folder that is ready to be hosted.
